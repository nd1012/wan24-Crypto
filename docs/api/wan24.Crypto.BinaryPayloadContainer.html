Class BinaryPayloadContainer

Binary payload container (you should extend this type and define the min./max. payload length in bytes in the MinPayloadLength and MaxPayloadLength properties (can be done by calling the protected constructors); default max. length is MaxValue, while the default min. length is 1 byte)
